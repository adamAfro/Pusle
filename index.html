Pusle

Proste puzzle typu: kilknij, żeby przesunąć w puste miejsce. Po poprawnym ułożeniu wywoływany jest event puslewon

Umieszczanie na stronie:

  1. Pobierz

    https://github.com/adamAfro/Pusle/releases

  2. Umieść

    <script src="sciezka/do/pusle-snipped.js"></script>

  3. Wkoduj

    let pusle = new Pusle.Game({
    (str) id, //id jakie ma mieć plansza
    (str) classes, //klasy planszy
    (obj) styles, //styl css planszy
    (str) html|text, //tekst lub html wewnątrz planszy - nie używać
    (HTMLElement) previous|parent, //element-rodzic lub element poprzedzający
    }, {
    (str) imageUrl, //ścieżka lub adres do obrazka
    (num) tileWidth, //szerokość płytki w jednostkach miary
    (num) width, //szerokość planszy w płytkach
    (str) unit, //jednostka miary
    });

    pusle.turnOn();

    Wszystkie argumenty są opcjonalne, ale bez rodzica lub elementu poprzedzającego nie zobaczysz pusli na stronie
